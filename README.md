# Heart Disease Prediction System

A machine learning web application that predicts the risk of heart disease using patient health information.

## 🚀 Features
- Predicts heart disease risk
- Built using Logistic Regression
- Interactive Streamlit UI
- Real-time predictions
- Probability score output

## 🛠 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## 📊 Machine Learning Model
- Logistic Regression
- Accuracy: 86.41%
- F1 Score: 87.92%

## 📁 Project Files
- `app.py` → Streamlit application
- `LR_heart.pkl` → Trained ML model
- `LR_scaler.pkl` → Feature scaler
- `LR_columns.pkl` → Feature columns

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
